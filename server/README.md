# Instalacion de node.js
```
Se realiza la busqueda en google de node js, y se selecciona el boton
"Descargar Node.js" y se selecciona la version LTS, y se descarga el archivo
```

```
Se ejecuta dandole permisos de si a todo lo solicitado por pantalla para que al terminar su instalacion entre a la terminal de git bash 
```

```
Se verifica su instalacion usando node -v la cual nos dice la version que tenemos instalada de node 
```

# Uso de json-server
```
Ingresa a la consola de git bash en la cual pondra npm install -g json-server para instalar de manera global el json-server este dara un mensaje similar al siguiente:
```

```
added 58 packages in 5s
18 packages are looking for funding
  run `npm fund` for details

```

```
ingresamos a una carpeta cualquiera en la cual creamos una carpeta old y dentro de esta carpeta old creamos una de nombre server ingresamos a server y desde la terminal realizamos code.
```

```
Se crea un archivo llamado db.json este contendra la informacion en formato json que requeramos
```

```
Se ejecuta el json-server con el siguiente comando dentro de la terminal de git bash npx json-server db.json para iniciarlizar el servidor de nuestro json
```
